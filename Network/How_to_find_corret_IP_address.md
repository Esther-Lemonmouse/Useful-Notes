# 我的Github为啥又上不去了？

首先，+86的Github直连起来比较慢，某些时刻干脆就连不上。原因也无需说明，这么多年了~

2021.7.16更新: 仨月多了，俺才发现github被搞了SNI封锁emmm，而且是不定时的。也就是说变成了间歇性正常...唉，底下的方法对这招是没用的。
